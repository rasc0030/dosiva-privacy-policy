Política de Privacidade — Dosiva

Última atualização: abril de 2026

Resumo: O Dosiva armazena todos os seus dados exclusivamente no seu dispositivo. Nenhuma informação é enviada para servidores externos, coletada por terceiros ou compartilhada com qualquer pessoa.

1. Dados coletados

O Dosiva permite que você registre informações de saúde para seu uso pessoal, incluindo:

  • Medicamentos, dosagens, horários e estoque
  • Registros de tomada de medicamentos
  • Medições de glicemia (glicose no sangue)
  • Locais de aplicação de insulina
  • Configurações de notificação e preferências do app

Todos esses dados são inseridos por você voluntariamente e existem apenas para ajudá-lo a gerenciar sua rotina de medicação.

2. Armazenamento

Seus dados são armazenados exclusivamente no seu dispositivo, em um banco de dados local (SQLite). O Dosiva não possui servidores, não transmite dados pela internet e não utiliza serviços de nuvem.

Se você desinstalar o app, os dados serão apagados permanentemente — a menos que você tenha feito um backup manual usando a função de exportação do app.

3. Compartilhamento de dados

O Dosiva não compartilha seus dados com terceiros. Nenhuma informação é enviada para anunciantes, parceiros comerciais, serviços de análise ou qualquer outra entidade.

O app possui uma função de exportação de relatório em PDF e backup, que você pode usar voluntariamente para compartilhar informações com seu médico ou salvar uma cópia de segurança. Essas ações são sempre iniciadas por você.

4. Permissões do dispositivo

O Dosiva solicita as seguintes permissões do Android, todas necessárias para seu funcionamento:

  • Notificações: Para enviar lembretes de medicação nos horários configurados
  • Alarme exato: Para garantir que os lembretes sejam disparados no momento correto
  • Vibração: Para alertas de notificação
  • Otimização de bateria: Para solicitar que o Android não restrinja os lembretes em segundo plano
  • Iniciar com o dispositivo: Para reagendar lembretes automaticamente após reiniciar o celular

Nenhuma dessas permissões é usada para coletar ou transmitir dados.

5. Serviços de terceiros

O Dosiva não integra serviços de terceiros como analytics, rastreamento, publicidade ou login social. O app funciona de forma completamente independente e offline.

6. Segurança

Como os dados ficam apenas no seu dispositivo, a segurança depende das proteções do próprio aparelho (senha, biometria, criptografia do sistema). Recomendamos manter seu dispositivo protegido com senha ou biometria.

7. Dados de crianças

O Dosiva não é direcionado a crianças menores de 13 anos. Se você é pai ou responsável e acredita que uma criança forneceu dados através do app, os dados podem ser apagados desinstalando o aplicativo.

8. Alterações nesta política

Esta política pode ser atualizada ocasionalmente. Alterações significativas serão comunicadas através de uma atualização do app. A data da última atualização está indicada no topo desta página.

9. Contato

Se você tiver dúvidas sobre esta política de privacidade ou sobre o app Dosiva, entre em contato pelo e-mail: dosiva.app@gmail.com
